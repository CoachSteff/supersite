# About Supersite

Supersite is an open-source, universal website template that combines modern web development with AI capabilities.

## Our Mission

To provide a powerful, easy-to-customize website template that anyone can use to create a professional web presence with integrated AI features.

## Technology Stack

- **Next.js 14**: React framework with App Router for modern web applications
- **TypeScript**: Type-safe development for better code quality
- **AI Integration**: Multi-provider support (Anthropic Claude, OpenAI GPT, Google Gemini, Ollama)
- **Markdown CMS**: Simple, file-based content management
- **YAML Configuration**: No-code customization for branding, features, and behavior

## Key Features

### Configuration-Driven Design
Everything can be customized through YAML files without touching code. Perfect for non-developers who want full control.

### AI-Powered
Built-in chat assistant that understands your content and helps visitors find information. Switch between providers easily.

### Modern Development
Built with cutting-edge web technologies and best practices. Fully tested with Jest and Playwright.

### Open Source

Supersite is open source and welcomes contributions from the community. Visit our GitHub repository to:
- Report issues
- Suggest features
- Contribute code
- Help with documentation
