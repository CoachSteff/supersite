# Welcome to the Supersite Blog

This is the first post on the Supersite demo blog. This blog demonstrates the built-in blogging capabilities of Supersite.

## Getting Started

Check out the [Quick Start guide](https://github.com/coachsteff/supersite/blob/main/docs/QUICKSTART.md) to get your own Supersite up and running in minutes.

## Blogging with Supersite

Creating blog posts is simple:

1. Create a markdown file in `content/blog/`
2. Use the naming convention: `YYYY-MM-DD-title.md`
3. Add frontmatter with title, date, author, and tags
4. Write your content in markdown
5. Your post appears automatically!
